Pada pertemuan mata kuliah Pemrograman Web ini, materi difokuskan pada penggunaan DOM (Document Object Model) dan JavaScript Vanilla sebagai dasar dalam membangun interaksi pada halaman web tanpa menggunakan framework tambahan.
Pembelajaran diawali dengan pemahaman bahwa website dibangun dari tiga komponen utama, yaitu HTML sebagai struktur, CSS sebagai tampilan, dan JavaScript sebagai logika. Dalam konteks ini, JavaScript digunakan untuk memanipulasi elemen HTML melalui DOM.
DOM memungkinkan kita untuk memilih (selector) dan mengakses elemen HTML, misalnya menggunakan getElementById atau querySelector. Setelah elemen berhasil diakses, kita dapat melakukan berbagai manipulasi seperti mengubah teks, nilai input, maupun atribut lainnya. Contohnya, nilai dari sebuah input dapat diambil menggunakan properti value.
Selain itu, dibahas juga tentang event listener, yaitu cara JavaScript merespons interaksi pengguna, seperti klik (onclick), mengetik (oninput), atau peristiwa lainnya. Event ini biasanya dihubungkan dengan fungsi (callback function) yang akan dijalankan saat event terjadi.
Materi juga mencakup proses validasi input menggunakan JavaScript, misalnya memastikan input tidak kosong atau sesuai format tertentu sebelum diproses lebih lanjut. Untuk membantu proses debugging, digunakan console.log untuk melihat data di console, serta alert untuk menampilkan pesan ke pengguna.
Di bagian akhir, dibahas sedikit mengenai optimasi dan penggunaan DOM secara efisien, termasuk penggunaan selector berbasis class atau id untuk mengambil elemen, serta pentingnya penulisan kode yang terstruktur dan mudah dipahami.
Dengan memahami DOM dan JavaScript Vanilla, mahasiswa diharapkan mampu membuat halaman web yang interaktif, responsif terhadap input pengguna, serta memiliki logika yang berjalan langsung di browser tanpa ketergantungan pada library tambahan
